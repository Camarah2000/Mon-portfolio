# Mon-portfolio
Description:
  Ce projet est un portfolio personnel que j'ai créé en utilisant HTML et CSS. L'objectif est de présenter mes projets, compétences et informations de contact de manière structurée et esthétique.
Technologies utilisées:
  HTML : Pour créer la structure de la page, organiser le texte, insérer des images et créer des liens.
  CSS : Pour styliser la page, ajouter des couleurs, des bordures, et créer des conteneurs pour structurer visuellement le contenu.
Fonctionnalités:
  Création de plusieurs pages HTML reliées entre elles par des liens de navigation.
  Mise en page responsive grâce à l'utilisation de conteneurs CSS.
  Personnalisation des polices, couleurs, marges et bordures pour une apparence soignée.
  Intégration d'images pour illustrer les projets présentés.
Améliorations futures:
  Ajouter des animations CSS pour rendre le site plus dynamique.
  Optimiser l'affichage sur mobile pour une meilleure expérience utilisateur.
  Ajouter des projets supplémentaires au portfolio.
